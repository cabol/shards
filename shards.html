Description

This is the main module, which contains all Shards API functions. This is a wrapper on top of shards_local and shards_dist.

Data Types

continuation()

continuation() = shards_local:continuation()

metadata()

metadata() = shards_local:state()

state()

state() = {LocalState::shards_local:state(), DistState::shards_dist:state()}

Defines the shards state:

  • LocalState: Local state handled by shards_local.
  • DistState: Distributed state handled by shards_dist.
